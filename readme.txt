Gutenberg block to apply css filters  to image 

== Description ==
This plugins lets you add image to page and post and apply css filters to the image

== Installation ==

This section describes how to install the plugin and get it working.


1. Upload the plugin files to the `/wp-content/plugins/image-effects` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Image effects block will appear in media section
4. Add image and apply effects through setting section of block
